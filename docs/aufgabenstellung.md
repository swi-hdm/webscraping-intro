## Aufgabe: Personal Scraper (2 Punkte)

Wählen Sie eine Webseite aus, die etwas mit einem persönlichen Hobby oder Interesse zu tun hat (z.B. eine Seite über Kochen, eine Fan-Seite für eine Band, ein Sportverein, ein Online-Shop für Sneaker, etc.). **Verboten sind**: Google, Wikipedia, Amazon, Facebook/Instagram.

### Das Skript muss die folgenden Anforderungen erfüllen:

**- Navigation**: Die Seite muss erfolgreich aufgerufen werden.

**- Interaktion (Der "Klick")**: Das Skript muss eine aktive Handlung ausführen. 

Beispiele:

* Einen Suchbegriff in eine Suchleiste eintippen und "Enter" drücken.
* Auf einen "Mehr laden" oder "Kategorie" Button klicken.
* Einen Cookie-Banner wegklicken (akzeptieren).

**- Extraktion**: Das Skript muss **zwei** unterschiedliche Informationen von der Seite in der Konsole ausgeben (z.B. Überschrift eines Artikels + Datum, oder Produktname + Preis).

**- Beweisfoto**: Das Skript muss automatisch einen Screenshot erstellen, nachdem die Interaktion stattgefunden hat. Benennen Sie die Datei `beweis_[IhrName].png`.

## Abgabe

- Die JavaScript-Datei `(.js)`.
- Der generierte Screenshot.
- Ein kurzer Kommentar im Code (1-2 Sätze), warum Sie diesen CSS-Selektor gewählt haben.

Alles bitte bis zur Deadline in eine ZIP-Datei packen und in Moodle hochladen!
