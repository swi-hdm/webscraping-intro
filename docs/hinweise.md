## Technische Aspekte

**Async / Await**: Das Web ist langsam. Wenn wir sagen "Öfnne eine Web-Page", dauert das Millisekunden oder Sekunden. **Await** sagt dem Code: "Warte hier, bis der Browser fertig geladen hat, bevor du die nächste Zeile ausführst."

**Selektoren**: Um dem Scraper zu sagen, welches Element er anklicken oder auslesen soll, nutzen wir CSS Selektoren. Tipp: Rechte Maustaste im Browser auf ein Element -> "Untersuchen" (Inspect). Dort sieht man die Klassen und IDs.
